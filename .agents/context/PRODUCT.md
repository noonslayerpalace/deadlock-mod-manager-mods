# Deadlock Mod Manager

Deadlock Mod Manager is a desktop utility for discovering, installing, updating,
and managing mods for Valve's Deadlock. The product is for players who want mod
management to feel reliable and reversible without manually editing game files.

## Users

- Players who install a small number of cosmetic or audio mods and need a clear,
  low-stress way to enable, disable, update, and remove them.
- Power users who maintain large mod libraries, multiple profiles, addon limits,
  local VPK files, and frequent updates.
- Contributors and maintainers who need to diagnose state drift, broken local
  files, profile imports, update flows, and support reports quickly.

## Product Principles

- Make operational state obvious: game readiness, update availability, download
  state, profile state, addon count, and VPK problems should be easy to spot.
- Keep destructive actions explicit and recoverable: deletes, repairs, resets,
  profile imports, and file operations need clear scope and confirmation.
- Prefer dense but scannable management UI over marketing layout. The app can
  feel stylish, but repeated workflows must stay fast.
- Preserve the Deadlock-adjacent atmosphere without hiding controls behind mood.
  Visual identity supports the tool; it does not replace clarity.
- Treat local file mismatch and addon analysis as repair workflows, not as app
  failure states.

## Anti-References

- Generic SaaS dashboards with oversized cards and decorative metrics.
- Modal-first repair flows that interrupt routine library management.
- Icon-only destructive controls without strong tooltip, label, or confirmation.
- Pages where completed history looks as urgent as active work.
- Monochrome dark UI where every surface has the same visual weight.

## Voice

Confident, concise, and practical. Error and repair copy should explain what
happened, what the app will do, and what files or mods are affected.
